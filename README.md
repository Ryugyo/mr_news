## Project description
Mr.News is a news aggregator website that compiles news feeds from a variety of online sources for the user. It is a lightweight website and can be operated by any kind of user, with or without programming skills. It is designed as a helping tool for students/seniors/business people to reach the fastest news when they donâ€™t wish to spend much time on browser different website pages. With this web application, users are able to organize, read and share information they are interested in.
